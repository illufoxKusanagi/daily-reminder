# Build Scripts

This directory contains helper scripts for building and running the application.

## Linux/macOS

### build-all.sh
Builds both frontend and backend:
```bash
chmod +x build-all.sh
./build-all.sh
```

### run-dev.sh
Runs the application in development mode:
```bash
chmod +x run-dev.sh
./run-dev.sh
```

## Windows

### build-all.bat
Builds both frontend and backend:
```cmd
build-all.bat
```

### run-dev.bat
Runs the application in development mode:
```cmd
run-dev.bat
```
